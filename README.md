# mvrt-website

MVRT's snazzy new website.

## Requirements

 + NodeJS
 + Grunt ``npm install -g grunt-cli``
 + Ruby
 + Bundler ``gem install bundler``
 + Bower ``npm install -g bower``

## Installation

```bash
$ git clone <git url>
$ bundle install && npm install && bower install
$ grunt
```
